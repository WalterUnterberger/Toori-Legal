Toori Privacy Policy

Effective date: May 14, 2026

About Toori

Toori is an educational Android application designed to support Japanese language learning, including kana, kanji, vocabulary study, progress tracking, and educational learning tools.

Information We Collect

Toori may collect the following information when you create and use an account:

  • Email address
  • Full name (if provided during registration)
  • Firebase user identifier (UID)
  • Learning progress and synced educational progress data
  • Password authentication credentials (handled securely by Firebase Authentication)

How We Use Your Data

  • User authentication and account management
  • Synchronizing learning progress across sessions
  • Providing core educational functionality
  • Account recovery and password reset support

Data Storage and Security

User account and synced progress data are stored using Google Firebase services. Data transmitted between the application and backend services is encrypted in transit.

Account Deletion

Users may delete their account and associated synced data at any time.

See the dedicated account deletion page: Delete Your Toori Account

Third-Party Services

Toori uses Google Firebase services for authentication and secure cloud data storage.
