Overview

This page provides legal, privacy, account deletion, and data licensing information for Toori, an educational Android application for learning Japanese.

Privacy Policy

Introduction

This Privacy Policy describes how Toori collects, uses, stores, and protects information when you use the App.

Toori supports Japanese language learning, including kana, kanji, vocabulary study, progress tracking, and related learning tools.

Information We Collect

  • Email address
  • Full name, if provided during registration
  • Firebase user identifier
  • Learning progress and study statistics
  • App usage timestamps related to learning activity

Passwords are not stored directly by Toori. Authentication is handled securely through Firebase Authentication.

How We Use Information

  • To provide and maintain app functionality
  • To authenticate users and manage accounts
  • To store and display learning progress
  • To synchronize learning progress where account sync is enabled
  • To support password reset and account recovery

Toori does not sell, rent, or trade personal information and does not use personal data for advertising.

Data Storage and Security

User account data and synced progress data are stored using Google Firebase services. Some learning data may also be stored locally on the user's device.

Third-Party Services

Toori uses Firebase services provided by Google LLC for authentication and backend functionality.

More information: Firebase Privacy and Security

Account Deletion

Users may delete their Toori account directly inside the App:

Profile → Settings → Delete Account

Deleting an account removes the user's authentication account, user profile information, and associated synced data stored for the account.

If you cannot access the App, you may request account deletion by email: w.unterberger.dev@gmail.com
